# 🏠 RentHub Marketplace

A modern peer-to-peer rental marketplace where users can list, discover, and rent items from each other.

## 🚀 Live Demo

**[https://renthub-marketplace.vercel.app/](https://renthub-marketplace.vercel.app/)**

## ✨ Features

- 🔐 Secure JWT-based authentication with Google OAuth
- 📦 List items for rent with images and pricing
- 💬 Real-time chat between renters and listers
- 🤖 AI-powered chatbot assistant
- 💳 Purchase invoice system
- 📱 Fully responsive design

## 🛠 Tech Stack

- **Frontend**: React + Vite
- **Backend**: Node.js + Express
- **Database**: MongoDB (Atlas)
- **Deployment**: Vercel

## 🏗 Getting Started

### Prerequisites
- Node.js 18+
- MongoDB connection string

### Installation

```bash
# Clone the repository
git clone https://github.com/shrii-maha/RENTHUB-.git
cd renthub-marketplace

# Install dependencies
npm install
cd server && npm install

# Set up environment variables
cp .env.example .env
# Fill in your values in .env

# Run development servers
npm run dev
```
